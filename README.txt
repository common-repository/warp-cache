== Description ==

Simple, light and powerful cache plugin.<br>
No customisation required, only activate plugin and plugin will do the rest.<br>
Integrated GooglePage Speed Insights for monitoring how well website is optimized.<br>
You can view how the plugin improve performance of your website through the plugin dashboard.<br>
Plugin offers you and other suggestions how you can improve speed of your website.

== Installation ==

This section describes how to install the plugin and get it working.

	1. Upload the plugin files to the /wp-content/plugins/warp-cache directory, or install the plugin through the WordPress plugins screen directly.
	2. Activate the plugin through the 'Plugins' screen in WordPress

You can view how the plugin improve performance of your website through the plugin dashboard.
	
	1. Go to Settings > Warp Cache
